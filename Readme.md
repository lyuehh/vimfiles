## Install

* `git clone git@github.com:lyuehh/vimfiles.git ~/.vimfiles/`

for Mac & Linux:

* `cd ~`
* `ln -s .vimfils/vim ./.vim`
* `ln -s .vimfiles/vimrc ./.vimrc`
* `git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
